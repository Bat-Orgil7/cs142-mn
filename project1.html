CS142 Лаборатори 1: HTML болон CSS

Дуусах: 2023, 2 сарын 15, Лхагва гараг 11:59 PM

Даалгавар 1 (20 оноо)

Ашиглагдаж буй CSS загвараасаа шалтгаалан хоёр өөр байдлаар харагдах боломжтой зөвхөн нэг HTML файл үүсгэ. Таны үүсгэсэн HTML файлын нэр нь index.html байх ба, “CS142 Project #1” гэсэн гарчигтай байна. Энэхүү HTML файлтай зэрэгцээ styleA.css болон styleB.css нэртэй хоёр загварууд үүсгэнэ үү.

A Загварын Шаардлагууд

  • Босоо эгнээнд зэрэгцүүлсэн зургаан ширхэг хайрцаг хэлбэртэй элемент байх ёстой.
  • Хайрцгууд хэвтээ чиглэлд голлож, босоо чиглэлд хоорондын зай нь тэнцүү байна. Вэб хөтчийн цонхны хэмжээг өөрчлөх үед хайрцагнууд хоорондын хэмжээ нь мөн дагаж өөрчлөгдөнө (дэлгэцийн босоо чиглэлийн дагуу хоорондын зай нь тэнцүү байхаар). Гэсэн хэдий ч, хайрцгууд нэг нэгэн дээрээ давхарлахгүй, мөн хэмжээгээ өөрчлөх ёсгүй.
  • Хайрцаг бүр 100x100 пиксел хэмжээтэй ба дээд талдаа 1px хүрээтэй (хүрээний өнгө: #687291) байх бөгөөд доторх үсэгнүүд өргөөшөө голлосон байна.
  • Хайрцагнуудын өнгө нь сөөлжилсөн байна. (өнгөнүүд: #dfe1e7, #eeeff2)
  • Сүүлчийн элементийн өнгө нь #687291 байх ба 4px хэмжээтэй хар хүрээтэй байна. Мөн үсэг нь уртаашаа голлож байрлана.
  • Бүх элементийн фонт нь Tahoma байх бөгөөд, 40 пиксел хэмжээтэй байна.

B Загварын Шаардлагууд

  • Дэлгэцний зүүн дээд булангаас эхлэн хэвтээ чиглэлд зэрэгцүүлсэн таван ширхэг хайрцаг хэлбэртэй элемент байна.
  • Вэб хөтчийн цонхны хэмжээг өөрчлөх үед хайрцгууд байрлалаа алдахгүй. (Жишээлвэл: Цонхны хэмжээ хэтэрхий жижиг үед бүх хайрцгийг дэлгэцэнд харуулж багтахгүй байсан ч А-аас Е хүртэл нэг эгнээнд хэвээрээ байна.)
  • Сүүлчийн хайрцаг цонхны баруун доод буланд байрлах бөгөөд цонхны хэмжээг өөрчлөх үед дагаж хөдлөнө.
  • Хайрцаг бүр 100x150 пикселийн хэмжээтэй (өнгө: #eeeff2) бөгөөд зүүн талдаа 10px өргөнтөй (өнгө: #D0D0FF) цэгэн хүрээтэй байх ба хайрцагнууд хоорондын зай 10px байна.
  • Хайрцган дээр хулганаа аваачих үед сум нь гарны хэлбэрт шилжих бөгөөд хайрцаг болон доторх үсэгний өнгөнүүд солигдоно. (хайрцагны өнгө: yellow, үсэгний өнгө: goldenrod)
  • Бүх элементийн фонт нь Tahoma байх бөгөөд, 40 пиксел хэмжээтэй байна.
  • Цэгэн хүрээ болон үсэгнүүдийн хооронд 10px зай байна.

Дээрх шаардлагуудад дурдагдаагүй хэмжээсүүдийг (Ж.нь: B загварын хайрцаг болон вэб хөтчийн цонхны хүрээ хоорондын зай г.м) та өөрөө тохируулан сонгоно уу.

Эмх цэгцтэй байдал (5 оноо)

Таны HTML файл XHTML 1.0 шаардлагад нийцэж байх бөгөөд, http://validator.w3.org дээрх шалгалтыг давах ёстой. Мөн дээрээс нь HTML болон CSS файлууд тань цэгцтэй, хялбар уншигдахуйц, мөрийн зай болон кодын бүтцийн хувьд зохистой байх тал дээр анхаарна уу.

Нэмэлт оноо (2 оноо)

Дараах CSS ойлголтууд дундаас хоёрыг нь сонгож styleC.css нэртэй гурав дахь загварт хэрэгжүүлснээр нэмэлт оноо авах боломжтой.

  • gradients
  • transitions
  • animations
  • web fonts
  • shadows

Хэрэглэгдэхүүн

  • Энэ хичээлийн хүрээнд гүйцэтгэх бүх лабораторийн ажилд Chrome вэб хөтчийг ашиглана уу. Ингэснээр вэб хөтчөөс шалтгаалах асуудлуудаас сэргийлнэ.
  • Даалгаврыг гүйцэтгэхдээ ямар ч Жаваскрипт болон бэлэн сангууд ашиглаж болохгүй. Кодоо зөвхөн өөрийн гараар бичнэ үү.

Ойртуулалт (hint)

  • Даалгаврыг гүйцэтгэхэд дараах CSS шинж чанарууд хэрэг болж магадгүй юм. (код дээрээ хэрхэн хэрэгжүүлэх талаар Google хайлт хийж судлана уу): 
    display: inline-block;
height: 100%;
white-space: nowrap;
  • Даалгаврын ядаж нэг хэсэгт CSS Flexbox байрлалыг ашиглах шаардлага гарах болно.

Даалгаврыг шалгуулах

Даалгавар 1-г хэрэгжүүлсэн index.html, styleA.css, styleB.css, болон (сонголтоор) styleC.css файлуудаа Teams сувгаар дамжуулан илгээх бөгөөд лабораторын цаг дээр багшид өөрийн биеэр асуугдаж оноогоо авна.
